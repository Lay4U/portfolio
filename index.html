PROJECT 08

차선 인식 시스템

멀티미디어처리기술 2019년 1학기

차량의 블랙박스 데이터를 활용하여 opencv와 CNN을 활용하여 차선을 인식하는 시스템을 개발하여 현재 차량이 차선을 벗어나고 있는지 확인하는 시스템을 개발하였다.

VIEW MORE
차선 인식 시스템을 개발하기 위하여 컴퓨터 과학과 4명이 조를 이루어서 프로젝트를 진행하였다. 매 프레임마다 Calibration을 위한 값을 새로 계산할 필요가 없었고 한번 계산한 값을 저장해 두기 위해 Pickle이라는 파이썬 모듈을 사용했다. 또한 카메라 왜곡을 보정하기 위해 OpenCV의 Camera Calibration을 사용하였고 Sliding WIndow Search를 위한 Binary Image를 만들기 위해 Filtering단계를 거쳤다 이때 필터는 Canny Filter을 사용하였다. 고정된 thresholding 값으로 필터링을 하면 차선이 인식이 안될때가 있어 Sobel Operator를 같이 사용하여 개선된 이미지를 얻었다. 차선을 인식하기 위하여 Region of Interest를 사용하였고 Masked된 이미지를 추출할 수 있었다. 또한 곡선 차선을 인식하기 위하여 Sliding Window Search Method를 이용하였다. Sliding Window로 인식한 차선 데이터를 사용자가 볼 수 있도록 원본의 이미지에 인식 결과를 표시하는 Superimposition을 적용하여서 프로젝트를 완성시켰다.
쉽게 생각했던 차선 인식은 의외로 매우 어려운 작업이였다. 많은 기하학적 지식을 요구하였고 이미지를 추출하는데 있어 최적의 조합을 구하는것이 매우 어려운 일이였다... 그래도 인터넷의 자료와 OpenCV를 기반으로 여러 조합을 찾고 개선해 나가면서 정확도를 올렸다. 많은 솔루션들이 자세하게 설명되어 있었고 이러한 솔루션들을 적용시켜 가면서 영상 처리를 위한 알고리즘을 이해하고 성능도 높이면서 차선을 인식할 수 있는 프로젝트를 성공적으로 완성 시킬 수 있었다.

PROJECT 09

실시간 채팅 프로그램

네트워크프로그래밍 2018년 1학기

소켓 프로그래밍을 기반으로 실시간 채팅 시스템을 구현하였다. 리눅스 환경에서 개발하였고 수업시간때 배운 네트워크 이론을 기반으로 irc 채팅 시스템을 구현하였다.

VIEW MORE
실시간 채팅 프로그램을 만들기 위해 4명의 컴퓨터과학과 학생들과 프로젝트를 진행하였다.
3-way handshake 이후 생성된 소켓의 번호를 실제 Address Family와 연결해주고 상대방의 접속을 기다린 후 접속을 수락한 뒤에 accept가 되면 통신을 실행한다. 소켓을 이용하여 포트 번호에 접속을 하고 send와 recv를 생성하여 동시에 메세지를 주고 받을수 있도록 만들었고 while문을 돌면서 input함수의 메세지를 작성하여 서버로 메세지를 보내며, 동시에 서버로부터 메세지를 지속적으로 받아 출력한다. 서버는 포트를 열고 while문을 돌며 클라이언트를 기다리고 클라이언트가 접속하면 list에 connection정보를 담는다. send 쓰레드가 생성되면 while문을 돌며 Recv 쓰레드에서 queue를 통해 받은 메세지들을 모든 클라이언트에게 보낸다. recv쓰레드가 실행되면 while문을 돌며 메세지를 받고 메세지를 보낸 상대의 connection정보와 queue를 이용해 send로 메세지를 보내는 방식으로 구현하였다.
